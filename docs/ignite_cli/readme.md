# Using SPN with Ignite CLI

These documents detail how to use `spn` with [Ignite CLI](https://github.com/ignite/cli).  `Ignite CLI` commands that 
connect to `spn` can be accessed using:
- `ignite n [command]`
- `ignite network [command]`
